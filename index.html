Audio Guided Maze Navigation

Kalkidan Asgedom  |  CMSC498F  |  Spring 2026

Can spatial audio alone support navigation in a virtual environment?

Abstraction

There is limited research on audio being used as the primary means of navigation. This project aims to explore the power of audio in low visibility environments where visual cues are not hevily reliable. This Audio Guided Maze Navigation project explores whether spatial audio alone can support sufficient navigation in a virtual reality environment.

Background

Humans naturally use sound and spatial awareness in their day to day lives. As one of our main five senses, sound is a way that we understand our environment, navigate through it, and protect ourselves. The main goal of XR systems, especially Virtual Reality technologies, is presence and immersion. Because of this XR systems are predominantly vision based and don't explore or leverage the power of audio as much as visual and sense ques.

Summary

This project set out to answer a simple but underexplored question: can spatial audio alone support navigation in a virtual environment? Using Unity and a Meta Quest 3 headset, a VR maze was built where the user must find the exit by following a directional sound source in a dark environment with no visual navigation aids.

The project successfully demonstrates that spatial audio can serve as a primary navigation tool in VR. Users are able to orient themselves and find the exit using only sound. The addition of footstep audio, haptic feedback at the exit, and atmospheric lighting further contributed to the sense of presence and immersion within the experience.

Building this project provided hands on experience with Unity, the XR Interaction Toolkit, 3D spatial audio configuration, and VR development for the Meta Quest 3. Moving forward this project lays the groundwork for more rigorous research into audio based navigation in XR with potential applications in accessibility, emergency evacuation systems, and cognitive research.
